# 🏥 Patient Wellness App

A comprehensive AI-powered health tracking and wellness management application built with React and TypeScript.

## 🌟 Features

### 📊 Dashboard
- Real-time health metrics overview (steps, heart rate, sleep, water intake)
- Smart health alerts and reminders
- Quick action buttons for common tasks

### 💓 Health Metrics Tracker
- Log blood pressure, glucose, weight, and temperature readings
- Historical data visualization
- AI-powered pattern detection and health insights
- Anomaly detection for early warning signs

### 💊 Medication Management
- Digital medication schedule with time-based reminders
- Track medication adherence
- Smart reminder system based on daily routines
- Support for multiple medications with different frequencies

### 📋 Personalized Wellness Plan
- Customized daily activity schedules
- Progress tracking with visual indicators
- AI-generated recommendations for:
  - Exercise adjustments based on health metrics
  - Dietary suggestions for glucose management
  - Sleep optimization strategies
- Health goal tracking with progress bars

## 🤖 AI-Powered Insights

The app uses simulated AI features to demonstrate:
- Pattern detection in health metrics (e.g., glucose spikes)
- Personalized recommendations based on individual health data
- Adaptive wellness plans that adjust to user behavior
- Predictive alerts before health issues escalate

## 🚀 Technology Stack

- **React 18** - Modern UI library
- **TypeScript** - Type-safe development
- **Vite** - Fast build tool and dev server
- **CSS3** - Responsive design with gradients and animations

## 📦 Installation

```bash
npm install
```

## 🏃 Running the App

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view in your browser.

## 🔨 Build for Production

```bash
npm run build
```
